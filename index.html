Khởi đầu với GitHub Pages

13/02/2026

Hướng dẫn tạo trang web tĩnh với GitHub Pages

GitHub Pages là một dịch vụ miễn phí cho phép bạn host các trang web tĩnh trực tiếp từ GitHub repository. Trong bài viết này, tôi sẽ hướng dẫn các bạn:

  • Tạo repository mới trên GitHub
  • Cấu hình GitHub Pages settings
  • Deploy website của bạn lên internet
  • Tùy chỉnh domain name (nếu có)

Note: Bạn có thể sử dụng HTML, CSS, JavaScript thuần hoặc các static site generators như Jekyll.

Nguyên tắc thiết kế web tối giản
